## 核心模块

| 模块 | 说明 |
|------|------|
| `multi_factor_model.py` | 96因子多因子选股模型 (V6.3)，23个大类，6波进化 |
| `x_god_strategy.py` | X大神4步闭环策略引擎（初筛→复审→红队→超越X融合） |
| `superpower_fusion.py` | Superpower x X-God 化学反应融合引擎 |
| `triple_engine_fusion.py` | APEX + DSA + FinRL 三引擎融合器 |
| `unified_fusion.py` | 统一融合引擎（96因子40% + 三引擎35% + 消息面25%） |
| `risk_engine.py` | 组合风控 + Kelly仓位 + ATR止损 + 压力测试 |
| `backtest_engine.py` | 完整回测框架（策略信号驱动） |
| `data_fetcher.py` | 统一数据获取层（快兰斯快讯 + 东方财富行情/K线） |
| `news_sentiment_engine.py` | 消息面情绪分析引擎（11行业 + 16个股映射） |
| `strategy_pipeline.py` | 统一策略流水线（串联7大引擎） |
| `auto_scheduler.py` | 自动化任务调度器（cron表达式） |
| `main.py` | 统一命令行入口 |

## 快速开始

```bash
# 安装依赖（仅标准库，无需额外安装）
# Python 3.8+

# 日频策略扫描
python main.py daily-scan --stocks 600519,300750,002594

# 单股深度分析
python main.py analyze --stock 600519

# 消息面监控（快兰斯实时快讯）
python main.py news

# 风控检查
python main.py risk --positions 600519:0.12,300750:0.08

# 策略回测
python main.py backtest --stocks 600519,300750 --days 120

# 启动自动化调度器
python main.py scheduler --duration 300
```

## 96因子体系

经过6波进化（28→42→56→72→88→96），覆盖23个大类：

- **价格动量** (5因子): 短期/中期/长期动量、动量加速度、动量反转
- **成交量** (5因子): 量比、量能趋势、量价背离、异常放量、缩量企稳
- **波动率** (4因子): 历史波动率、波动率聚类、波动率突破、隐含波动率偏离
- **技术指标** (8因子): MACD/KDJ/RSI/BOLL/布林带宽/ATR/OBV/DMI
- **均线系统** (5因子): 多头排列、空头排列、均线发散、均线缠绕、均线斜率
- **K线形态** (4因子): 锤子线/吞没/十字星/启明星
- **资金流向** (5因子): 主力净流入/大单净比/资金流向趋势/北向资金/融资融券
- **基本面** (6因子): PE/PB/ROE/营收增长/净利润增长/毛利率
- **市场情绪** (5因子): 恐慌指数/换手率/涨跌停比/新高新低比/市场宽度
- **行业轮动** (4因子): 行业相对强度/行业资金流向/行业动量/行业估值
- **消息面** (3因子): 新闻情感/社交媒体热度/分析师共识
- **周级别趋势** (8因子): 周动量收敛/周量价共振/周趋势持续性等
- **更多因子**: 龙虎榜/大宗交易/股东增减持/筹码分布/突破因子/反转因子/共振因子/自适应因子/Alpha因子/Beta因子/风格因子/流动性因子

## 技术特点

- **纯标准库**: 无第三方依赖，零安装成本
- **懒加载架构**: 避免循环依赖，按需加载模块
- **双源数据**: 快兰斯快讯 + 东方财富行情，自动 fallback
- **7阶段流水线**: 数据→情绪→因子→策略→融合→风控→仓位
- **自动化调度**: 4个预置任务（日扫描/周复盘/月初筛/预测验证）
